# prep-html-script-tag-quiz-notes

## Quiz Questions

Answer the following questions in the provided markdown file before turning in this exercise:

- What tag is needed to load JavaScript scripts into an HTML document?
  A <script> tag

- How do you use a script tag to write JavaScript directly in the HTML document?
  You place the JavaScript (either directly or via link and src attribute) between <script></script>, and it can go either in the <head> or <body> section depending on how you want it to run.

- How do you use a script tag to load an external JavaScript file?
  Use the src attribute along with the script tag and link to the external file
